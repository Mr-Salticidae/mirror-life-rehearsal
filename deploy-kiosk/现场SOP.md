# 展台现场 SOP（打印成 A5 贴在展台机旁）

## 开摊（3 步）

1. 开机，等桌面完全加载
2. 双击 **启动展台.bat**，等全屏画面出现（约 10 秒）
3. （可选，RTX 报告卖点）确认 LM Studio / Ollama 已自动启动并加载模型；
   没有也不影响——报告自动走离线模板，观众无感知

## 收摊（2 步）

1. 双击 **停止展台.bat**（会自动关掉全屏浏览器和后台服务）
2. 关机

## 出现任何异常

**重启机器再开摊 = 恢复 90% 的问题。** 程序自带看门狗：
游戏卡死 180 秒自动回待机，无人操作 120 秒自动回待机，崩溃自动恢复。

| 症状 | 处理 |
|---|---|
| 屏幕白屏/黑屏不动 | 键盘按 F5；无效则重启机器 |
| 全屏没了露出桌面 | 双击 启动展台.bat（不会重复起服务，只补开浏览器） |
| 没有声音 | 用手点一下屏幕任意处；仍无声找技术同事 |
| 观众说扫码打不开 | 让观众用手机流量（不连展馆 WiFi）重试 |

## 紧急止血（仅当技术同事电话指示时操作）

- 哨兵线（打枪游戏）玩不了：双击 **standby\切换_下线哨兵线.bat**，然后屏幕按 F5
- 恢复：双击 **standby\切换_恢复五线.bat**，然后屏幕按 F5
