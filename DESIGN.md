# 《镜像自我·人生预演》v2 —— BW 展位互动影游

> 36h 急活 · 对标《底特律：成为人类》· GenJi 展位 · RTX 本地 AI
> 定案：从零新建 / AI剧照+电影化动效 / RTX本地AI报告=结局彩蛋 / 展位离线运行

## 一句话

玩家在一面「镜子」里预演一次人生：三个年龄节点做出选择，走向军人 / 画家 / 赛车手
三种职业，各自进入一段可玩的迷你游戏，最后由本地 AI 生成专属《人生预演报告》。

## 为什么这版不普通（对比被否的初版）

| 初版（被否） | v2 |
|---|---|
| 问卷 + 3D 背景 | 章节化影游：剧照运镜 + 字幕 + 限时抉择 |
| 选择只影响报告文本 | 选择改变剧情走向，最终进入**完全不同的游戏类型** |
| 无可玩性 | 三个迷你游戏（FPS / 涂鸦 / 竞速），围观者看得懂、排队者想上手 |
| 无「错过感」 | 底特律式**分支流程图**：你没走的人生灰着、锁着，勾人重玩 |
| 实时 3D 观赏度上限低 | MJ 电影级剧照 + 视差推镜 + 胶片颗粒，观赏度上限=图的上限 |

## 体验流程（单人 4~6 分钟）

```
待机吸引模式（大屏循环，任意键进入）
  └→ 序幕「镜面」 点亮仪式，1 次点击
  └→ 第一章「七岁」   2 个抉择节点
  └→ 章间分支图（3s 自动，可跳过）
  └→ 第二章「十七岁」 2 个抉择节点（含 QTE 倒计时）
  └→ 章间分支图
  └→ 第三章「二十五岁·镜前」 最终确认 / 换门
  └→ 职业前奏剧照 + 操作说明
  └→ 迷你游戏 60~90s（军人→FPS靶场 / 画家→涂鸦墙 / 赛车手→夜环线）
  └→ 终幕：RTX 本地 AI《人生预演报告》+ 全分支图 + 分享卡
  └→ 「重新预演」→ 回待机
```

## 分支系统

隐藏倾向值（底特律式暗数值）：`勇`(军人) / `彩`(画家) / `驰`(赛车手)。
每个选择 +1~+2；第三章展示当前最高倾向的「预演人生」，允许最后逆转（三扇门）。
第二章 D 节点的「妥协」不加值但记 `遗憾` flag，会写进 AI 报告 —— 展位话题点。

| 节点 | 场景 | 选项 → 倾向 |
|---|---|---|

| C 十七岁·天台 | 高三天台黄昏 | 美术集训单→彩 / 军校报名表→勇 / 卡丁车场兼职→驰 |

## 美术方向

- **主角永不露正脸**：全部剧照用背影 / 侧影 / 局部特写 —— 一致性零风险、性别中立、
  契合「镜像自我」主题（你看不到脸，因为那是你）。唯一的"正脸"是终幕镜中发光的模糊轮廓。
- 风格锚：cinematic film still、变形宽银幕、teal-orange 电影调色、35mm 颗粒、16:9。
- 网页端表现：Ken Burns 慢推镜 + 鼠标微视差 + letterbox 黑边 + 胶片颗粒层 + 光斑转场。
- 出图管线：MJ（v8.2 `--preview` 写实增强），prompt 清单见 `assets/PROMPTS.md`；
  未出图时程序化占位图保证全流程可跑可验收。

## 迷你游戏（各 60~90s，键鼠）

1. **军人《守夜》**：Three.js pointer-lock 夜间靶场；标靶弹出射击计分，平民靶误伤扣分。
2. **画家《一面墙》**：canvas 喷漆（按住喷涂/换色/飞溅粒子），限时自由创作，
   完成后作品**合成进结局剧照**——"你的画留在了这座城市"。
3. **赛车手《夜环线》**：Three.js 雨夜公路，←→/A D 转向躲车流拾氮气，距离计分。

## RTX 本地 AI 报告（结局彩蛋）

- OpenAI 兼容端点，`public/config.json` 可配（默认 `http://127.0.0.1:1234/v1`，LM Studio；
  兼容 Ollama）。输入：选择路径 + 遗憾flag + 游戏成绩。
- 输出：称号 + 三段人生叙事 + 一句给现实的你。canvas 导出分享卡。
- **兜底**：端点不可用 → 本地模板引擎按路径拼装报告，体验不中断。

## 展会稳定性

- 全资源本地打包，零外网请求（AI 端点是局域网/本机）。
- 闲置 90s 自动回待机吸引模式；错误边界兜底回待机；场景切换释放 Three.js 资源。
- 目标机：展位 RTX 机器，1080p/4K 大屏，Chrome/Edge kiosk 全屏。

## 技术栈

Vite + React 18 + TypeScript + zustand + three。无其他运行时依赖。静态构建，本地静态服务即可跑。
