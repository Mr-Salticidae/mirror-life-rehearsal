## 1. 今晚新改动·重点测区（优先级最高）

| # | 测什么 | 怎么测 | 预期 |
|---|---|---|---|
| R1 | **游戏时长 20s** | 五条线各进游戏一次 | 倒计时从 20 起；到 0 正常结算出评级 |
| R2 | **评级线手感** | 每个游戏认真玩 2–3 局 | 认真玩≈A、玩得好=S、摆烂=B；哪个游戏"太容易S/根本没法S"记下来附得分 |
| R3 | **二段高潮** | 守夜 11s 后 / 节奏连击 15·30 / 对接窗口 / 赛车最后 2.2s | 演出触发且不突兀；普通人打不打得出来记录实际数值 |
| R4 | **守夜持枪** | 进守夜转动视角、开枪 | 右下角手枪不穿帮、开枪有后座回弹；帧率无明显下降 |
| R5 | **分支图无？？？** | 走完整叙事，看章末分支图+报告页小图 | 未走分支只有灰虚线+空心点，无"？？？"文字 |
| R6 | **四轴优势侧** | 报告页四轴 + 保存的海报 | 金条在占比高的一侧、优势字高亮；海报微缩条标签=优势项 |
| R7 | **动态剧照**（素材落位后） | 走叙事/职业前奏 | 有视频的场景动起来、无视频的静图正常；视频构图与静图一致不跳变 |
| R8 | **海报变体**（素材落位后） | 同一职业用不同路线打两遍 | 不同路线出不同结局图与"XX型"标题；没放变体图时用基础图不报错 |
| R9 | **BGM**（落位后） | 全流程听一遍 | 循环无接缝爆音；进游戏自动变轻、结束恢复；静音按钮有效 |

## 2. 全量回归矩阵（重点测区过完后扫一遍）

每条职业线走一遍完整流程，逐项打勾：

```
线路：soldier / painter / racer / musician / astronaut（每线一份）
[ ] 待机页 → 按住镜面 1.2s 进入
[ ] 8 节点叙事：点选 / 计时QTE超时 / 长按卡（故意各触发一次）
[ ] 章末分支图正常显示并 5s 自动推进（或点"继续"）
[ ] 职业前奏 → 进入游戏 → 20s 完赛 → 评级动画
[ ] 报告页：流式文字 / 称号 / 四轴 / 时间线 / 镜子没给你的人生
[ ] 保存分享卡：能下载、图上信息与本局一致
[ ] 再试一次：只重开游戏；重新预演：回待机
[ ] 五扇门故意选与倾向不符的门 → 称号带"换过门的·"
```

稳定性专项（任测一线时顺带）：

```
[ ] 闲置 120s 自动回待机
[ ] 游戏中途刷新页面 → 重新进入不报错
[ ] 守夜点"跳过考核" → 直达报告不卡死
[ ] 静音开关全程有效
[ ] 窗口缩放/全屏切换后画面不错位
```

## 3. 已知非 bug（别再报）

- 报告页显示 OFFLINE MODE / 内容是模板文风 —— 内测站无本地 AI，**这就是展会断网态**
- `?career=` 直跳后报告页无人生轨迹图，显示"调试直达·无叙事轨迹" —— 设计如此
- 海报右下二维码位是"镜"字印章占位 —— 指向待定
- 分支图未走分支看不到选项文字 —— 主导定案（防剧透）

## 4. 上报模板（一条 bug 一份，发群里或写进 `docs/测试反馈_日期.md` 推上来）

```
【编号】T-01（按发现顺序自编）
【优先级】🔴阻断（流程走不下去）/ 🟡影响体验 / ⚪观感小问题
【环境】内测站 or 本地 / 浏览器 / 分辨率
【线路与位置】如：racer 线 · 夜环线游戏中
【复现步骤】1. … 2. … 3.（能稳定复现写"必现"，偶发写出现概率）
【预期】应该怎样
【实际】实际怎样（附截图/录屏，录屏优先）
【附加】得分/评级等数值（评级手感类必填）
```

数值手感类反馈（R2/R3）不用截图，直接列：`游戏名 / 认真程度 / 得分 / 评级 / 一句话感受`，
每人每游戏至少 2 条——这批数据今晚就要用于校准评级线。
