# 《镜像自我·人生预演》技术文档

> 2026-07-07 夜 · 面向技术人员 · 基准版本：main `f381df2`（项目收尾终版）
> 仓库：[Mr-Salticidae/mirror-life-rehearsal](https://github.com/Mr-Salticidae/mirror-life-rehearsal)（私有）
> 内测站：https://mr-salticidae.github.io/mirror-life-rehearsal-preview/ （同时是扫码手机报告页宿主，BW 结束前不可下线）

## 一、定位与形态

BW（BILIBILI WORLD）展位互动影游：观众在展位机器上用约 3–5 分钟"预演一次人生"——
8 个叙事抉择节点 → 分支图复盘 → 进入对应职业的 20s 迷你游戏 → 本地 AI 生成人生预演报告 + 分享海报，
并可**扫码在自己手机上带走完整图文报告**、**现场打印**（A4 卡片 / 210mm 卷纸完整报告两种版式）。
单页 Web 应用，全程可离线运行（断网是展会常态，也是设计前提——含二维码生成在内零外网依赖）。

## 二、技术栈

| 层 | 选型 | 版本 |
|---|---|---|
| UI | React + TypeScript | 18.3 / 5.5 |
| 构建 | Vite | 5.x（`tsc -b && vite build`，产物纯静态，**双入口**：`index.html` 展台 + `r.html` 手机报告页） |
| 3D | Three.js | 0.166（WebGL，三个 3D 迷你游戏 + GLB 资产） |
| 2D | Canvas 2D | 节奏游戏、涂鸦墙、分享海报合成 |
| 状态 | zustand | 4.5（单 store 状态机） |
| AI | 本地 LLM（LM Studio，OpenAI 兼容端点） | SSE 流式；离线模板兜底 |
| 二维码 | qrcode | 1.5（打进构建，纯离线生成；jsQR 仅 devDep 做解码回归） |
| 压缩 | 浏览器原生 CompressionStream | deflate-raw；老浏览器自动走未压缩回退 |
| 音频 | Web Audio API | 程序化音效（纯噪声系）+ BGM `audio/bgm.mp3` |

无后端、无数据库、无外网依赖；构建产物即部署单元。

## 三、运行时架构

### 3.1 状态机（store.ts + App.tsx）

```
attract(待机/人生墙) → prologue(擦亮镜面) → story(叙事节点×8, 按章分组)
  ↘ 每章末 interlude(分支图人生曲线, 5s 自动推进)
→ career-intro(职业开场) → game(20s 迷你游戏) → report(AI 报告+海报+扫码/打印)
  ↘ 「再试一次」只重开游戏刷分 / 「重新预演」全量回待机
```

- 闲置 120s 自动回待机（展位无人值守）。
- 每幕结束整场渐隐至纯黑 0.9s 再切场（电影式谢幕）。
- 运维直达：`/?career=soldier|painter|racer|musician|astronaut` 跳过叙事直进职业线（调试/展位应急）。
- DEV 态暴露 `__game` / `__shareUrl` QA 钩子（自动化测试用）。

### 3.2 叙事引擎（story.ts）

- 8 节点（A,B,F,C,G,D,H,E）× 每节点 2–4 选项；选项带 `effect` 累积五维倾向值
  （guard/create/swift/rhythm/far ↔ 军人/画家/赛车手/音乐人/宇航员）。
- 交互形态：普通点选、计时 QTE（超时有专属文案与后果）、长按抉择卡（settled 哨兵防双触发）。
- 终局：E 节点五扇门自选职业；与倾向值最高项不一致时标记「换过门的」进称号系统。

### 3.3 分支图「人生曲线」（FlowChart.tsx）

- 底特律式复盘：一条 Catmull-Rom 连续金线（张力 1/6）只跟随选中项游动，不折回主轴，
  终点直接游进结局框；未走分支灰虚线+空心圆点（**不显示文案与「？？？」**，主导定案）。
- interlude（章末）与 final（报告页）共用渲染，仅布局模式不同。

### 3.4 迷你游戏（games/，统一 20s）

| 游戏 | 职业 | 技术 | 二段高潮 | 操作兜底 |
|---|---|---|---|---|
| 守夜（FPS 靶场） | 军人 | three.js + pointer lock | 11s 起平民靶大量混入 | 无锁降级瞄准 + 跳过出口 |
| 夜环线（三车道躲避） | 赛车手 | three.js + Kenney GLB 车模 | 末 2.2s 冲刺慢镜 | 点按屏幕左/右半变道 |
| 躁动（4 轨节奏） | 音乐人 | Canvas 2D + WebAudio 程序化鼓组 | 连击 15/30 升档炸屏 | 点击轨道 |
| 突围（纵向飞行射击 AstroBlaster） | 宇航员 | three.js（程序化几何 + flight 贴图增强） | 10s 碎片风暴 + 14s BOSS | 指针拖拽移动（键鼠双持） |
| 一面墙（喷漆涂鸦） | 画家 | Canvas 2D | 完成时"城市亮灯" | 纯指针操作，可提前完成 |

- **宇航员线已由一维对接《失重》整体换装为纵向 STG《突围》**（danbao757 实现）：
  飞船底部巡航+自动连射，陨石/无人机/拦截机/精英/BOSS 五类目标，P/S/H/B 拾取，
  3 格船体+受击无敌帧，连击倍率；对象池+圆形碰撞；星空/地球复用原 Docking 程序化做法，
  陨石/拾取/爆炸叠加 `sprites/flight/` 素材包贴图（载入前程序化几何兜底）。
- 守夜出靶做最小间距 3.5 排斥采样（最多 12 次重抽兜底），多靶同屏不叠位。
- 评级 S/A/B（lib/titles.ts 统一阈值表，20s 版已按比例重校，flight 档 S300/A180；待真人彩排微调）；
  评级联动称号池（S/A/B × 5 职业 + 「带着遗憾的·」「换过门的·」前缀）。
- 守夜持枪视图：同事手枪 FBX 经 `app/scripts/fbx2glb.mjs` 离线清洗
  （剔 C4D 场景杂物、烘焙立起旋转、归一化）转 GLB，挂相机右下带后座回弹。

### 3.5 AI 报告链路（lib/ai.ts）

- 在线：本地 LM Studio SSE 逐字流式；协议为**纯文本**（空行分段 + 「——」结语），JSON 仅兼容回退。
- 超时策略：停滞式（首包 20s / 块间 15s），非总时长。
- 离线：模板库（5 职业 × child/teen/job/word 段）+ 模拟流式（~2.3s），与在线观感一致。
- **推理遥测（RTX 算力可见化）**：流式过程采集真实数据——模型名/生成字数/首字到完稿耗时/字每秒，
  报告页 RTX LOCAL AI 标签下显示"模型 · N字/X.Xs · N字/秒 · 零云端请求"；
  仅真 AI 态返回，离线模板态自动隐藏。按"字/秒"计（端点不回传 token 数，不估算不编数字，诚实原则）。
  待机页配套断网徽标："⬤已离线·本地AI独立运行中"呼吸绿徽（online/offline 事件驱动），
  支撑展台"拔网线时刻"演示动线。
- 报告页四轴（GPTI 式人格）：由五维倾向值算 4 组对立占比，**按占比高左低右排列**，
  金条锚定优势侧+优势端高亮；TYPE 代码（如 GNTR）由四轴优势端字母拼成。
- 分享海报：Canvas 三段式合成（称号+评级印章 / 剧照+slogan / 四轴+结语+二维码），
  **2x 渲染输出 2160×2880** JPG（文字剧照同步锐化，二维码模块防糊）。

### 3.6 扫码带走链路（lib/share.ts + src/mobile/）

观众扫海报或屏幕上的二维码，在自己手机上打开完整图文报告——展台侧纯离线：

- **数据通道**：报告数据（SharePayload：职业/称号/评级/五维分/时间线/AI 文本）→
  原生 `CompressionStream('deflate-raw')` 压缩 → base64url → URL 片段 →
  `{reportBase}/r.html#<payload>`。编码首字符区分 `'1'`=压缩 / `'0'`=原文（老浏览器兜底）。
- **载荷瘦身**：离线模板报告只带种子（~400 字符，手机端用同一模板重新生成叙事文本）；
  真 AI 报告文本唯一，全文上车（~1KB）。
- **手机页**：`r.html` 为第二构建入口（src/mobile/），随内测预览站部署；
  与展台共用 `lib/reportModel.ts` 推导层（四轴/印记/未走人生），剧照复用站上 stills/；
  坏码/空链接有提示页。域名用 `config.json` 的 `reportBase` 覆盖（换部署时改这一处）。
- **二维码渲染**（防"扫不出"实案）：离屏按每模块整数设备像素绘制 → 关平滑贴入
  （零抗锯齿灰边，自动感知 ctx 缩放）；静区 4 模块（标准值）；ECC L；
  海报框位 176px（2x 画布下每模块 ≥2 整数像素），报告侧栏另有 150px 大码卡（现场扫屏）；
  生成失败回退镜印章。已用 jsQR 对导出成图做机器解码回归
  （模板载荷全图直解+裁剪解、真 AI 最坏用例 1150 字符 URL + JPEG 往返均逐字符一致）。

### 3.7 现场打印（components/PrintReport.tsx）

报告页「打印卡片」「打印完整报告」双按钮，双版式：

- **卡片**：分享海报 A4（210×297）整页满幅，零边距，上下同底色补满无白边。
- **完整报告**：210mm 卷纸版面（与手机扫码版同规格：称号/剧照/叙事/时间线/四轴/镜外人生/二维码/落款），
  **按内容长度动态注入 `@page` 页长**（实测版面高度取整+2mm，min-height 撑满防尾部露白），
  上限 1.2m；超长按 1.2m 分页且背景连续无缝。
- 实现：打印版面常驻屏外，`html[data-print]` 切换出图版面，打完自动复位。
- 展台配套：kiosk Chrome 加 `--kiosk-printing` 静默直印；打印驱动选无边距/连续纸模式。

### 3.8 资产管线

- **静态剧照**：19 张 MJ 生成图按节点 ID 归档（1920 宽 JPG，总 4.8MB），待机页全量预热+已载缓存首帧直出；
  缺图自动走程序化占位图。
- **结局变体图**：20 张 `end_{career}_{trait}.jpg`（5 职业 × 4 次强倾向，MJ V8.1 提示词包 v2 产出，
  112~338KB/张）——报告页按次强倾向自动选变体，缺图回退基础结局图（随放随生效）。
- **动态剧照**：13 条 Seedance 图生视频 `videos/{id}.mp4`（8 节点+5 职业开场；1080p30/去音轨/crf24，
  124MB→13MB）。懒探测不预热（进场景时按需检查），存在即叠在静图上**只播一遍定格尾帧**
  （无 loop，防 5s 素材跳跃重播；key 重建保证换幕重新起播），缺失/失败静图兜底。
- **3D 模型**：GLB 统一经 `lib/models.ts`（GLTFLoader + 缓存 + 失败返 null，游戏保留程序化几何兜底）；
  FBX 来料用 `app/scripts/fbx2glb.mjs` 清洗（用法：`node scripts/fbx2glb.mjs <src.fbx> <out.glb>`，
  需在 app 目录下运行；剔除清单在脚本内按名单匹配）。
- **像素/贴图**：守夜立牌靶序列帧首帧按 alpha 求包围盒紧致裁切；突围用 `sprites/flight/` 素材包。
- **音频**：BGM `audio/bgm.mp3`（Nocturnal Reverie，~3.5MB，播放/闪避/静音链路在跑）；
  程序化音效做过**减法**——剥除全部振荡器哔哔层（square/sawtooth/高频短音），只留噪声质感音
  （whoosh/spray/crash/ignite/heartbeat 系），hover/tick 直接删除；BGM 闪避 0.1→0.2 承担更多氛围。

## 四、稳定性护栏（展会级）

1. 错误边界包裹全应用；WebGL 创建失败 → 跳过游戏直达报告（主流程不断）。
2. AI 断线/超时 → 无缝切离线模板；缺图 → 占位图；动态剧照缺失 → 静图；BGM 缺失 → 静默跳过。
3. 二维码生成失败 → 回退镜印章（海报版式不破）。
4. pointer lock 被拒/不可用 → 无锁瞄准降级；每个纯键盘操作均有鼠标/触屏兜底。
5. 闲置 120s 回待机；「跳过考核」出口防卡死整条线。

## 六、工程实践

- 验证铁律：UI 交互回归必须 `document.elementFromPoint` 命中测试或原生视口真实坐标点击——
  程序化 `click()` 会绕过浏览器命中测试，漏检"看得见点不到"的遮挡缺陷（有实案）。
- 装饰性覆盖层（剧照背景、letterbox 黑边）一律 `pointer-events: none`。
- 二维码/海报类导出物验收要**机器解码回归**（jsQR 对成图直解），肉眼"看着像码"不算过。
- 遥测/演示数据诚实原则：只展示链路真实采集值，拿不到就不显示，不估算不编数字。
- 协作流：内容组走 `content/` + 分支 + PR（文风铁律见 COLLABORATION.md）；代码可直推 main，
  push 被拒先 `fetch + rebase`。
- 接续入口：仓库根 `PROGRESS.md` 第五节待办（每次阶段存档更新）。
