## 一、项目是什么

为 BILIBILI WORLD 展位打造的**互动影游**：观众用 3–5 分钟"预演一次人生"。

体验流程（玩家视角）：

1. **擦亮镜面**开始预演 → 从七岁的巷口出发，经历 **8 个人生抉择**（作文课上写下什么、天台上的少年心事、二十二岁的出租屋、二十五岁的镜前……）
2. 每章结束，一条**金色人生曲线**在分支图上游过你选的路——没走的路留作虚线
3. 走进五扇门之一：**军人 / 画家 / 赛车手 / 音乐人 / 宇航员**，玩一局 20 秒的职业迷你游戏（守夜靶场 / 喷漆涂鸦 / 雨夜飙车 / 节奏接拍 / 太空对接）
4. 本地 AI 逐字生成你的**人生预演报告**：专属称号 + 评级 + 四维人格轴 + 人生时间线 + "镜子没给你的人生"
5. 一键生成**分享海报**带走

技术形态：纯前端单页应用，本地 AI（RTX + LM Studio），**全程可断网运行**——断网时自动切离线模板，观感一致。
