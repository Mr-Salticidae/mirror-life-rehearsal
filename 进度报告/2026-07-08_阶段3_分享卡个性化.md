# 阶段性进度报告 ③ · 分享卡个性化完善

> 2026-07-08 晚 · 阶段 3/4 · 状态：**已完成，构建零报错**（成图观感在阶段 4 全流程回归中实拍验证）

## 一句话结论

观众带走的分享海报在原有"称号 + 类型代码 + 结局剧照 + 四轴 + 二维码"基础上，
新增四类个性化元素——**每一张海报现在都能讲出"这个人今晚走过的路"**，
且五条职业线的海报一眼可分。

## 新增的个性化元素

1. **职业专属配色点缀**：海报顶部的 TYPE 类型代码与标题分隔线，改用当前结局职业的
   调色板高光色（哨兵=冷月蓝 / 画家=霓虹紫 / 赛车手=雨夜蓝 / 音乐人=荧光紫 / 宇航员=星舷蓝）。
   此前五条线的海报顶部完全同色，现在职业身份从第一眼的色彩就开始表达。
2. **人生时间线带**：剧照带与四轴条之间新增一行"走过的地点"——
   巷口 · 作文课 · …… 八站按剧情序串成一条线。两位观众即使同结局同称号，
   时间线也会显出各自不同的人生路径。超长自动收间距/缩字号，保证一行放下；
   调试直达（无叙事轨迹）时自动隐藏。
3. **档案编号 + 日期戳**：左下角新增 `档案 MLR-20260708-XXXX` 编号行，
   每张海报独一无二，收藏感/晒图辨识度都更强。
4. **AI 溯源行**：真 AI 生成的报告，海报落款处会带上
   「RTX 本地 AI 实时生成 · XX 字/秒 · 零云端请求」实测遥测（与展台"算力可见化"口径一致）；
   离线模板态则标注「离线预演档案」——两态都诚实可查，不编数字。
